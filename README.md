# ADB Battery Monitor

A Python + pytest based automation project that monitors Android device battery
using ADB commands.

## Tech Stack
- Python
- Pytest
- Android Debug Bridge (ADB)

## Project Structure
- src/     : Core logic for ADB interaction
- tests/   : Pytest-based test cases
- reports/ : Test reports (generated)
- adb_logs/: ADB command logs (generated)

## Use Case
- Read battery level from connected Android device
- Validate battery drain or charging behavior
- Useful for QA automation and device health monitoring
